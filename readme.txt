Easily add Bugsnag error monitoring for your front-end.


== Description ==

This plugin adds front-end error monitoring for your WordPress installation. It monitors all javascript files from plugins, themes and WordPress core. If you do not have a Bugsnag account, simply signup at [Bugsnag.com](https://bugsnag.com).

== Installation ==

Bugsnag is available through the [WordPress Plugins Directory](http://wordpress.org/plugins/), so you can download and install automatically from the Plugins menu on your WordPress admin page.

To manually install this plugin:

1. Unzip `bugsnag-js.zip`
1. Upload the `bugsnag-js` directory from the zip file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click the 'Settings' link next to the Bugsnag plugin, and follow the instructions
